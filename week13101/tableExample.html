Language Fact
JavaScript JavaScript was created by Brandon Eich over a week during the holidays.
Python Python was created by Guido van Rossum over a weekend and was first released in 1991.
C++ C++ was developed by Bjarne Stroutstup at Bells labs.
Java Java was developed by James Gosling at Sun Microsystems and first released in 1995.
HTML HTML was created by Tim Berners-Lee in 1991 while he was working at CERN.
